# Proxying

If you have a web server you will need to do this if you want to use your domain.

If you have a domain and make a A record that points to our proxy server`173.208.244.4` the proxy ip is required along with the proxy status being off like in the photo below if you are using Cloudflare.

Then [make a ticket](https://billing.wavehost.org/submitticket.php) on the billing panel to proxy.

Don't have a Domain? Well you can use a free one from us `{name}.dev-web.club`   [https://dev-web.club/](https://dev-web.club/) Is a free subdomain that anyone can use for people who don't have a domain. [Make a ticket ](https://billing.wavehost.org/submitticket.php)if you would like to use our free subdomain!
